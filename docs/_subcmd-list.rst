git flow |BRANCH| list - List |BRANCH| branches
======================================================


Synopsis
-----------

``git flow |BRANCH| list`` [-h] [-v]


Description
-----------

List all existing |BRANCH| branches in the local repository.


Options
-----------

  -h, --help          Show help message and exit.
  -v, --verbose  Be verbose (more output).
